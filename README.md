# MRRG-Mobile

MRRG Mobile is the Android client for the MRRG business management platform.

It is designed for field workers to view assigned jobs, upload before/after photos and mark jobs as completed directly from site.

The app consumes the existing MRRG Spring Boot REST API.

Backend API

This mobile app requires the MRRG backend to be running:

http://10.0.2.2:4000
